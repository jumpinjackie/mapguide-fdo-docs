content
Glossary

For a resource, the content is XML data that is stored in the repository. Different types of resource use different schemas.

Item Schema For Content
Repository RepositoryContent schema
Application definition resource ApplicationDefinition schema
Drawing source resource DrawingSource schema
Feature source resource FeatureSource schema
Layer definition resource LayerDefinition schema
Load procedure resource LoadProcedure schema
Map definition resource MapDefinition schema
Print layout resource PrintLayout schema
Symbol definition resource SymbolDefinition schema
Symbol library resource SymbolLibrary schema
Web layout resource WebLayout schema
