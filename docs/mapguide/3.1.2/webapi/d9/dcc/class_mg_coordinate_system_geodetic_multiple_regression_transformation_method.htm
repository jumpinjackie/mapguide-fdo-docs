Detailed Description

This class defines the currently supported transformation methods for multiple regression geodetic transformations as they can be read via the MgCoordinateSystemGeodeticTransformDefDictionary. That is, the constants defined below are only valid for MgCoordinateSystemGeodeticTransformDef instances of type MgCoordinateSystemGeodeticTransformDefType.MultipleRegression.
