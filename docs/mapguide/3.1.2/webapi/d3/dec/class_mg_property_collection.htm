Detailed Description

MgPropertyCollection represents a collection of properties. All properties contained in an instance of a property collection are objects whose base class is MgProperty and whose derived class is one of the classes referenced in the topic Feature Properties . A call to a property object's GetPropertyType() method returns one of the integer constants defined in class MgPropertyType. This collection does not allow duplicate key names.

Note:
Collections are not thread safe.
