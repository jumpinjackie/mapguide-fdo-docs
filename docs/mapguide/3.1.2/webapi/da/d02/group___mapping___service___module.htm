Detailed Description

The Mapping Service allows you to display your feature data as layers in a map. Use it to create eMaps , designed for online viewing, or ePlots , designed for hardcopy printing.

The Mapping Service only supports Internet Explorer on Windows. For information on how to display information on other browsers and other platforms, see the Rendering Service .
